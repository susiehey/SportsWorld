SportsWorld API Documentation

This is the API documentation for the SportsWorld application.

Athletes

GET /api/athletes - Hent alle utøvere
GET /api/athletes/{id} - Hent én utøver
GET /api/athletes/byname?name=XYZ - Søk etter navn
GET /api/athletes/unpurchased - Hent utøvere som ikke er kjøpt
POST /api/athletes - Opprett ny utøver
PUT /api/athletes/{id} - Oppdater utøver
DELETE /api/athletes/{id} - Slett utøver
POST /api/athletes/{id}/purchase - Kjøp en utøver

Bilder vises via: /images/<filnavn>

Finance

GET /api/finance - Hent status
POST /api/finance/loan - Øk lånet
