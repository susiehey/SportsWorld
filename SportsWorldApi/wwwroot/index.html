SportsWorld API Documentation

This is the API documentation for the SportsWorld web-application.

MMA Athletes

Get all athletes: GET /api/athletes

Get athlete by id: GET /api/athletes/{id}

Register new athlete: POST /api/athletes

Update athlete: PUT /api/athletes/{id}

Delete athlete: DELETE /api/athletes/{id}

Upload athlete image: POST /api/imageupload

Images are stored in /images/firstname_lastname.png (Example: /images/valentina_shevchenko.png)

One Athlete

  • Id: int
  • Name: string
  • Gender: string
  • Price: int
  • Image: string
  • PurchaseStatus: bool

Finance

Get financial data: GET /api/finance

Increase money: POST /api/finance/increaseMoney
