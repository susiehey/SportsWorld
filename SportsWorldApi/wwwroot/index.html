SportsWorld API Documentation

This is the API documentation for the SportsWorld application.

Athletes

GET /api/athletes - Get all athletes
GET /api/athletes/{id} - Get one spesific athlete by id
GET /api/athletes/byname?name=XYZ - Search by name
GET /api/athletes/unpurchased - Get athletes that are not purchased yet
POST /api/athletes - Register new athlete
PUT /api/athletes/{id} - Update existing athlete
DELETE /api/athletes/{id} - Delete athlete
POST /api/athletes/{id}/purchase - Buy athlete

Pictures of athletes are shown here: /images/<filename>

Example: /images/valentina_shevchenko.jpg

Finance

GET /api/finance - Get financial status
POST /api/finance/loan - Loan money
